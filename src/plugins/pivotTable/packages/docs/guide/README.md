
# Introduction

## What is VuePivottable?

It is a Vue port of the jQuery-based [PivotTable.js](https://pivottable.js.org/)

* [David-Desmaisons/vue-plotly](https://github.com/David-Desmaisons/vue-plotly) - vue wrapper for plotly.js
